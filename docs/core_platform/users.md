# Users Management

Gestione utenti del sistema.

---

## Funzioni

lista utenti  
scheda utente  
creazione utente  
disattivazione utente  
reset password  

---

## Campi utente

name  
email  
department  
role  
api_key  
active  

---

## Departments

I reparti disponibili sono:

quality  
administration  
production  
managing  

Ogni utente deve appartenere a uno di questi reparti.

---

## Regole

Solo admin può creare utenti.
